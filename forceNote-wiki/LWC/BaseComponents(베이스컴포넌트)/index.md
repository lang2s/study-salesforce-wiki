# BaseComponents(베이스컴포넌트) — 로컬 인덱스

## 파일 목록

| 파일 | 컴포넌트 | 한 줄 요약 |
|---|---|---|
| [[lightning-accordion]] | `lightning-accordion` | 접고 펼 수 있는 아코디언 — 섹션 토글, 다중 열기, 프로그래밍 열기 |
| [[lightning-tabset]] | `lightning-tabset` | 탭 그룹 컨테이너 — ontabchange, variant(default/scoped/vertical), active-tab-value |
| [[lightning-input]] | `lightning-input` | 다목적 입력 필드 — type 변형 14가지, 유효성 검사, change·commit 이벤트 |
| [[lightning-combobox]] | `lightning-combobox` | 단일 선택 드롭다운 — options 배열, onchange, 읽기 전용 입력 |
| [[lightning-datatable]] | `lightning-datatable` | 정렬·선택·인라인 편집 테이블 — columns 정의, 커스텀 타입, 행 액션 |
| [[lightning-modal]] | `lightning-modal` | 모달 다이얼로그 — LightningModal 상속, open/close, 반환값 |

| [[lightning-record-picker]] | `lightning-record-picker` | 레코드 검색·선택 위젯 — filter, clearSelection, 다중 선택 패턴 |
| [[lightning-button]] | `lightning-button` 외 5종 | 버튼 패밀리 — button·button-icon·button-menu·button-group·stateful |
| [[lightning-card]] | `lightning-card` | 카드 컨테이너 — title, icon-name, actions/footer 슬롯 |
| [[lightning-spinner]] | `lightning-spinner` | 로딩 스피너 — size, variant, lwc:if 조건 렌더링 패턴 |
