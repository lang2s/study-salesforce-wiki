# runtime_industries_cpq Namespace

> 제품 검색 및 제품·카탈로그·카테고리 관리를 위한 Industries CPQ Apex 네임스페이스 (managed package)

> [!warning] 이 노트는 Apex Reference Guide의 네임스페이스 설명 기반으로 작성된 스텁입니다. 클래스·메서드 상세 API는 **runtime_industries_cpq namespace** 공식 문서를 참조하세요.

---

## 개요

`runtime_industries_cpq` 네임스페이스는 **제품 검색 및 제품·카탈로그·카테고리 관리** 클래스와 메서드를 제공한다.

- managed package 기반 네임스페이스 (`runtime_` prefix)
- 표준 Apex Reference Guide가 아닌 패키지 전용 문서에 API 공개
- Industries CPQ(Configure, Price, Quote) 시나리오에서 사용
